Vercel

Den här webbplatsen har publicerats på Vercel via GitHub. För att publicera en webbplats på Vercel kan man skapa ett gratiskonto med inloggning via GitHub. Sedan klickar man på Add new -> Project (sök rätt på det repo som ska publiceras från en lista) -> Import -> Deploy. Det går även att lägga till en domän och ändra sökväg till webbplatsen. Med Vercels gratisplan (Hobby) kan man publicera ett limiterat antal webbplatser. Det går även att uppgradera till betalplaner på olika nivåer (Pro och Enterprise) för en mer krävande publicering. Steget efter gratisversionen (Pro) kostar $20/månad. Vercel stöder programmeringsspråken JavaScript/TypeScript (NodeJs), PHP, Phython, Ruby, Rust och Go.
