Bootstrap

Bootstrap är ett populärt CSS-ramverk som går att installera via npm eller CDN. I den här uppgiften har CDN använts. Fördelen med Bootstrap är att ramverket förenklar arbetet, främst när det gäller responisvitet. Tabellen på den här sidan blir scrollbar i sidled på en liten skärm och en brytpunkt har lagts vid 1200 px (XL). En nackdel med Bootstrap är att det endast finns ett litet utbud av färger att välja på för färdiga komponenter som till exempel knappar. Det går dock att använda SASS för att skapa ett eget färgschema. Nedan finns två länkar, en till Bootstraps officiella sida och en till en instruktionsvideo från MDBootstrap.com som jag fann hjälpsam.
